# Papers Slide structure


```
Title
|--- Summary
|--- Introduce and define terms 
|--- Motivation & Background of this paper
|--- Approach -> Explain details in this paper
|--- Contrbution they made
|--- Wrap up (Conclusion).
```


## Latex

```latex

\section{Paper Title}
\begin{frame}{Summary}
% Briefly explain the paper here.    
\end{frame}


%%% [Optional] %%%
\begin{frame}{Terms}
% Define tearms here if needed.
\end{frame}

\end{frame}

\end{frame}

\end{frame}
```
